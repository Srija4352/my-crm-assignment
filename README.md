# AI-First CRM - HCP Module

This project is an AI-first Customer Relationship Management (CRM) system designed for the Healthcare Professional (HCP) module. 

## Features
- **Log Interaction Screen**: Allows users to log interactions with HCPs via a structured form or a conversational chat interface.
- **AI Integration**: Powered by LangGraph and Groq's `gemma2-9b-it` model for intelligent interaction management.
- **Frontend**: React UI with Redux for state management.
- **Backend**: Python FastAPI.

## How to Run
1. **Clone the repository**: `git clone https://github.com/Srija4352/my-crm-assignment.git`
2. **Setup Backend**: Navigate to `backend/`, install requirements (`pip install -r requirements.txt`), and run the server.
3. **Setup Frontend**: Navigate to `frontend/`, install dependencies (`npm install`), and start the application (`npm start`).

## Technologies
- React, Redux, FastAPI, LangGraph, Groq LLM.
